┌──────────────────────────────────────────────────────────────────────────┐

┌──────────────────────────────────────────────────────────────────────────┐
│                       /api/employees                                      │

│  • GET    /status/{status}→ Get employees by status                     │
│  • GET    /search?q=      → Search employees                            │

================================================================================
                            DATA FLOW EXAMPLE
================================================================================

Example: Create New Employee

┌─────────────┐
│   React     │  1. User fills form and clicks "Create Employee"
│  Frontend   │
└──────┬──────┘
       │ 2. POST request to /api/employees
       │    Body: { firstName: "John", lastName: "Doe", ... }
       ▼
┌──────────────────┐
│  Employee        │  3. Receives request
│  Controller      │  4. Validates data
└──────┬───────────┘
       │ 5. Calls service method
       ▼
┌──────────────────┐
│  Employee        │  6. Generates ID: EMP016
│  Service         │  7. Creates Employee object
└──────┬───────────┘  8. Stores in ConcurrentHashMap
       │
       ▼
┌──────────────────┐
│  In-Memory       │  9. employees.put("EMP016", employee)
│  Storage         │
└──────┬───────────┘
       │ 10. Returns created employee
       ▼
┌──────────────────┐
│  Controller      │  11. Returns HTTP 201 Created
└──────┬───────────┘      with employee JSON
       │
       ▼
┌──────────────────┐
│   React          │  12. Receives response
│   Frontend       │  13. Updates UI with new employee
└──────────────────┘

================================================================================
                           DEVELOPMENT WORKFLOW
================================================================================

  1. Frontend Request    ──┐
                           ├──> CORS Check (Allowed)
  2. Hit REST Endpoint   ──┤
                           └──> Route to Controller
  3. Controller          ──┐
                           ├──> Validate Request
  4. Call Service        ──┤
                           └──> Execute Business Logic
  5. Service Layer       ──┐
                           ├──> Process Data
  6. In-Memory Storage   ──┤
                           └──> Read/Write ConcurrentHashMap
  7. Return Data         ──┐
                           ├──> Format Response
  8. Send Response       ──┤
                           └──> HTTP Status Code + JSON
  9. Frontend Receives   ──┐
                           └──> Update UI
